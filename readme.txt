== Description ==

This is a great new way for you as a blogger to earn some extra money.
SnoLooken is a plugin for coupling blog images to products in web shops, using the service [SnoLooken](http://Snolooken.se Snolooken)

Note that you need to [register](http://snolooken.se/user/newapplication) before you can use SnoLooken.

In it's current version this product is mainly usable for the Swedish audience.

You can find a demo of the functionality on [blogg.snolooken.se](http://blogg.snolooken.se)

== Installation ==

Use Wordpress' own installation process in the admin guide.
